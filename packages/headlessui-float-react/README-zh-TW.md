[English](README.md) | 繁體中文

* 💙 輕鬆整合 Headless UI & Tailwind CSS
* 💬 Floating UI (新版 Popper.js) 定位浮動元素
* 🔔 自動更新浮動元素
* ♾️ 支援 Transition
* 🚪 支援 Portal
* ➡️ 支援箭頭 (Arrow)
